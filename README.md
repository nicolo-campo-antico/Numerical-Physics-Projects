# Numerical Physics Projects

> A collection of numerical simulations and computational models developed during the **Engineering Physics MSc** at *Politecnico di Milano*.

Each folder corresponds to a graduate-level course and contains one or more self-contained projects, combining physical theory with numerical implementation in Python. 

---

## Repository Structure

```
Numerical-Physics-Projects/
├── Artificial-Neural-Networks/
    └──  HAR/
├── Dynamical-Systems/       
└── SPoCS/ 
    └──  Ising_2D/                      
```

---


## Tech Stack

| Tool | Role |
|------|------|
| Python 3 | Primary language |
| NumPy / SciPy | Numerical computation |
| Matplotlib | Visualization |
| TensorFlow / PyTorch | Neural network models |

---

## Getting Started

Clone the repository and navigate to any project folder:

```bash
git clone https://github.com/nicolo-campo-antico/Numerical-Physics-Projects.git
cd Numerical-Physics-Projects/<project-folder>
```

Each project folder contains its own `README.md` with specific instructions, dependencies, and results.

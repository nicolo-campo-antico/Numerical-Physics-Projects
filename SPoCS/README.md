# Statistical Physics of Complex Systems
In this folder you find the projects somehow related in topic with the SPoCS course from Polimi. By now only Ising 2D has been implemented. These are personal projects inspired by simple pedagogic simulations seen in class and physical subjects explored during the course — the goal is to go beyond the classroom, building clean and visual simulations from scratch that capture the essential physics.

**Implemented:**
- Ising 2D — Monte Carlo simulation of the 2D Ising model with real-time animation and observable computation (energy, magnetisation, specific heat, susceptibility, Binder cumulant)

**Planned:**
- Ising 3D
- Percolation theory
- Random walks and diffusion
